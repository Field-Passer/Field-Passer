## 1. 프로젝트 요약
- 실내 체육 시설 환불 시 기간에 따른 수수료 차등 반환
- 현재 실내 체육 시설 양도 시스템은 사용자의 접근성이 좋지 않음
- 실내 체육 시설 양도의 불편함을 이번 프로젝트를 통해 해결하고자 함

## 2. 프로젝트 설정
